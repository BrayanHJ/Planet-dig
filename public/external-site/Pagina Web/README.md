# Planetario Digital Chimalhuacán

Sitio web informativo y visualmente atractivo para el Planetario Digital Chimalhuacán.

## Descripción general

Este proyecto contiene varias carpetas de recursos (assets) y scripts para el funcionamiento del sitio. Cada carpeta importante dentro de `assets` incluye un archivo `CONFIG.md` que explica:
- El propósito de la carpeta
- El formato y tamaño recomendado de las imágenes
- Cómo modificar o agregar recursos
- Ejemplos de uso en el código

> **Revisa los archivos `CONFIG.md` en cada subcarpeta de `assets` para instrucciones específicas sobre imágenes, iconos, actividades, salas, etc.**

## Estructura principal

- `index.html` — Página principal del sitio
- `assets/` — Recursos (imágenes, iconos, actividades, salas, etc.)
    - Cada subcarpeta contiene un `CONFIG.md` con instrucciones
- `Scripts/` — Archivos JavaScript para la interactividad
- `styles/` — Hojas de estilo CSS

## Características

- **Diseño responsivo y moderno** con animaciones CSS personalizadas
- Navegación intuitiva y secciones bien diferenciadas
- Carrusel de funciones y horarios interactivo
- Optimización visual para imágenes y recursos

## Tecnologías

- HTML5, CSS3 (Flexbox, animaciones, transiciones)
- JavaScript vanilla (para interactividad)
- Google Fonts

## Mejoras futuras

- Adaptación completa a dispositivos móviles
- Refactorización a componentes reutilizables
- Implementación de variables CSS y preprocesadores
- Accesibilidad mejorada

## ¿Cómo ejecutar?

1. Descarga o clona este repositorio.
2. Abre `index.html` en tu navegador.
3. Consulta los archivos `CONFIG.md` en cada carpeta de `assets` para personalizar imágenes y recursos.
